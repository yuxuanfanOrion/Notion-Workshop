# Notion 块到 Markdown 格式映射

**[English](./format-mapping.md)** | **[中文](./format-mapping_cn.md)**

本文档展示 Notion 块如何转换为 Markdown 格式。

## 文本块

| Notion 块 | Markdown 输出 |
|-----------|---------------|
| 段落 | 纯文本 |
| 一级标题 | `# 标题` |
| 二级标题 | `## 标题` |
| 三级标题 | `### 标题` |

## 折叠标题

可折叠标题（带有可展开内容的标题）转换为 HTML `<details>` 元素：

**Notion：**
```
▶ 折叠一级标题
  └─ 子内容
```

**Markdown：**
```html
<details><summary>

# 折叠一级标题

</summary>

子内容

</details>
```

## 列表

| Notion 块 | Markdown 输出 |
|-----------|---------------|
| 无序列表 | `- 项目` |
| 有序列表 | `1. 项目` |
| 待办事项（未完成） | `- [ ] 任务` |
| 待办事项（已完成） | `- [x] 任务` |

嵌套列表项每层缩进 2 个空格。

## 折叠块

**Notion：**
```
▶ 折叠摘要
  └─ 隐藏内容
```

**Markdown：**
```html
<details><summary>折叠摘要</summary>

隐藏内容

</details>
```

## 代码块

**Notion：**
```
代码块，语言：javascript
console.log("hello");
```

**Markdown：**
````markdown
```javascript
console.log("hello");
```
````

## 引用和标注

| Notion 块 | Markdown 输出 |
|-----------|---------------|
| 引用 | `> 引用文本` |
| 标注 | `> 💡 标注文本`（带表情图标） |

## 其他块

| Notion 块 | Markdown 输出 |
|-----------|---------------|
| 分隔线 | `---` |
| 图片 | `![image](url)` |
| 书签 | `[url](url)` |
| 链接预览 | `[url](url)` |

## 表格

**Notion：**
```
| 列1 | 列2 |
| A   | B   |
| C   | D   |
```

**Markdown：**
```markdown
| 列1 | 列2 |
| --- | --- |
| A | B |
| C | D |
```

## 行内格式

| Notion 样式 | Markdown 输出 |
|-------------|---------------|
| **粗体** | `**粗体**` |
| *斜体* | `*斜体*` |
| ~~删除线~~ | `~~删除线~~` |
| `代码` | `` `代码` `` |
| [链接](url) | `[链接](url)` |

## 特殊块

| Notion 块 | 行为 |
|-----------|------|
| 分栏 | 内容合并为行内 |
| 同步块 | 获取原始内容 |
| 子页面 | 跳过（单独文件） |
| 子数据库 | 跳过 |

## 不支持的块

以下块会被记录但不会转换：
- 嵌入
- 视频
- 音频
- 文件
- PDF
- 公式
- 面包屑
- 目录
- 模板按钮
